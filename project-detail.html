AI-Powered Lane Detection System

A real-time lane detection system for autonomous vehicles, deployed on a Jetson Nano.

Role: Lead Developer
Timeline: 3 Months
Tech: Python, OpenCV, TensorRT

About The Project

This project was developed to create a robust and efficient lane detection algorithm capable of running on edge devices with limited computational power. The primary goal was to achieve high accuracy and real-time performance, making it suitable for autonomous ground vehicles (AGVs). The system processes a live video feed, identifies lane markings, and calculates the vehicle's position relative to the lane, providing crucial data for navigation.

Challenges & Solutions

One of the biggest challenges was performance optimization for the NVIDIA Jetson Nano. The initial Python and OpenCV implementation was too slow. To solve this, the deep learning model was converted and optimized using NVIDIA's TensorRT, which significantly accelerated inference speed. Another challenge was handling difficult lighting conditions and worn-out lane markings. This was addressed by implementing advanced image pre-processing techniques, such as adaptive thresholding and perspective transforms, to enhance the visibility of the lanes before feeding the frames to the model.
